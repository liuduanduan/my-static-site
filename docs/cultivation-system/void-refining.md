# 炼虚 / Void Refining

## 描述

炼虚常与空间、虚无、法则理解和更抽象的力量有关，不再只是灵力多少的比拼。

## 特征

- 更常涉及空间术法或领域
- 力量来自理解，而不仅是积累
- 在低阶世界中极为稀少

## 叙事作用

炼虚让修炼体系开始显得宇宙化、概念化。
