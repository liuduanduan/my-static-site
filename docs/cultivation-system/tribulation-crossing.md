# 渡劫期 · Tribulation Transcendence (Dùjié Qī)

<div class="entry-layout"><main class="entry-main">

## 摘要 · Summary

**中文：** 渡劫期属于中境界，寿元参考为十万年以上。度过天劫即可飞升仙界，是凡界通往仙界的门槛。

**English:** Tribulation Transcendence belongs to the Middle Realm. Lifespan reference: over one hundred thousand years. Surviving heavenly tribulation opens the path toward the immortal realm.

## 描述 · Description

**中文：** 这一境界是十四级修炼阶梯中的关键节点，会改变修士储存力量、理解大道、面对冲突和接触世界的方式。

**English:** This realm is one step in the fourteen-realm ladder. It changes how cultivators store power, understand the Dao, survive conflict, and interact with the world.

## 核心特征 · Key Characteristics

- **境界层级 / Realm Tier:** 中境界 / Middle Realm
- **寿元 / Lifespan:** 十万年以上
- **力量变化 / Power Shift:** 度过天劫即可飞升仙界，是凡界通往仙界的门槛。
- **叙事作用 / Narrative Role:** 标志能力、身份和风险进入新的门槛。

## 突破条件 · Breakthrough Conditions

**中文：** 积累圆满、因果清明，并在天劫中守住道心。

**English:** Reach completion, settle karma, and preserve Dao heart under tribulation.
