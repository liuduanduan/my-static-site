## 描述

结丹是修士把精气神与灵力凝聚为[金丹](/glossary/golden-core)的阶段，实力和寿命通常大幅提升。

## 特征

- 战力与寿命明显增长
- 金丹品质可能影响后续潜力
- 可以驾驭更强法宝并拥有更高宗门地位

## 叙事作用

结丹常表示主角真正成为一方强者。
