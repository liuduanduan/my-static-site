# 元婴 / Nascent Soul

<div class="term-meta">

</div>

## 描述

元婴期修士形成强大的灵性存在。在许多设定中，即便肉身被毁，元婴也可能逃离或夺舍。

## 特征

- 神识大幅增强
- 神魂类术法和分身能力更常见
- 可成为长老、老祖或区域强者

## 叙事作用

元婴把危险从肉身层面推进到神魂层面。
