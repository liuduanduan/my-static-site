# 合体 / Body Integration

## 描述

合体把此前积累的肉身、神魂、法力和道悟整合成更完整的修炼体系。

## 特征

- 肉身强度与恢复能力大增
- 物质力量与精神力量更协调
- 更能承受高阶环境与法则压力

## 叙事作用

合体常为凡界最终冲突和飞升前考验做铺垫。
