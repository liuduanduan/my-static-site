# 炼气期 · Qi Refining (Liànqì Qī)

<div class="entry-layout"><main class="entry-main">

## 摘要 · Summary

**中文：** 炼气期属于下境界，寿元参考为100-120岁。灵气为气态，共有多个小境界，重点在感应与引气入体。

**English:** Qi Refining belongs to the Lower Realm. Lifespan reference: 100-120 years. Qi remains gaseous; the focus is sensing Qi and drawing it into the body.

## 描述 · Description

**中文：** 这一境界是十四级修炼阶梯中的关键节点，会改变修士储存力量、理解大道、面对冲突和接触世界的方式。

**English:** This realm is one step in the fourteen-realm ladder. It changes how cultivators store power, understand the Dao, survive conflict, and interact with the world.

## 核心特征 · Key Characteristics

- **境界层级 / Realm Tier:** 下境界 / Lower Realm
- **寿元 / Lifespan:** 100-120岁
- **力量变化 / Power Shift:** 灵气为气态，共有多个小境界，重点在感应与引气入体。
- **叙事作用 / Narrative Role:** 标志能力、身份和风险进入新的门槛。

## 突破条件 · Breakthrough Conditions

**中文：** 积累稳定灵气，打通经脉，建立可持续运转的基础周天。

**English:** Accumulate stable Qi, open meridians, and build a sustainable circulation.
