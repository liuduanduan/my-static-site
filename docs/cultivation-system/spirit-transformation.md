# 化神期 · Spirit Severing / Transformation (Huàshén Qī)

<div class="entry-layout"><main class="entry-main">

## 摘要 · Summary

**中文：** 化神期属于下境界，寿元参考为数千岁。元婴化神，开始触及更高天地法则。

**English:** Spirit Severing / Transformation belongs to the Lower Realm. Lifespan reference: several thousand years. The Nascent Soul transforms into spirit and begins touching higher laws.

## 描述 · Description

**中文：** 这一境界是十四级修炼阶梯中的关键节点，会改变修士储存力量、理解大道、面对冲突和接触世界的方式。

**English:** This realm is one step in the fourteen-realm ladder. It changes how cultivators store power, understand the Dao, survive conflict, and interact with the world.

## 核心特征 · Key Characteristics

- **境界层级 / Realm Tier:** 下境界 / Lower Realm
- **寿元 / Lifespan:** 数千岁
- **力量变化 / Power Shift:** 元婴化神，开始触及更高天地法则。
- **叙事作用 / Narrative Role:** 标志能力、身份和风险进入新的门槛。

## 突破条件 · Breakthrough Conditions

**中文：** 完成神魂升华，理解自身大道方向。

**English:** Refine the soul and clarify the direction of one own Dao.
