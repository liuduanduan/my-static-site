# 化神 / Spirit Transformation

<div class="term-meta">
  <div><strong>中文</strong><br />化神</div>
  <div><strong>拼音</strong><br />hua shen</div>
  <div><strong>常见位置</strong><br />第五境</div>
  <div><strong>核心主题</strong><br />神魂与神识蜕变。</div>
</div>

## 描述

化神进一步强化修士的精神、神识和对高阶力量的感知，常代表从地方强者迈向世界级存在。

## 特征

- 神识范围和精度增强
- 更重视法则、领域或道意
- 心魔和心境问题更突出

## 叙事作用

故事常从宗门争斗转向古迹、秘境和老怪物之间的较量。
