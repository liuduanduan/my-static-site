## 特殊道路

- 散仙可能是在飞升失败后保留修为的特殊存在。
- 地仙、天仙、真仙等可作为成仙后的细分。
- 炼体、剑修、魔修、丹道等道路可能改变通常的进阶重点。

## 阅读提示

这套顺序适合作为翻译和阅读参考，不代表所有小说的统一设定。
