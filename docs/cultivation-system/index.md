# 修炼体系 · Cultivation System

本页整理从炼气期到道祖境的十四级境界阶梯，方便读者、译者、创作者和游戏策划快速对照。

## 境界总览图 · Realm Timeline

<div class="realm-timeline">
<a class="realm-step" href="/cultivation-system/qi-refining"><span class="mark">1</span><span><h3>炼气期 / Qi Refining</h3><p>下境界 · 100-120岁 · 灵气为气态，共有多个小境界，重点在感应与引气入体。</p></span></a>
<a class="realm-step" href="/cultivation-system/foundation-building"><span class="mark">2</span><span><h3>筑基期 / Foundation Establishment</h3><p>下境界 · 约200岁 · 灵气由气态转为液态，修士真正踏入修仙大门。</p></span></a>
<a class="realm-step" href="/cultivation-system/core-formation"><span class="mark">3</span><span><h3>结丹期 / 金丹期 / Core Formation / Golden Core</h3><p>下境界 · 500-800岁 · 液态灵力凝为固态金丹，是修仙道路的重要分水岭。</p></span></a>
<a class="realm-step" href="/cultivation-system/nascent-soul"><span class="mark">4</span><span><h3>元婴期 / Nascent Soul</h3><p>下境界 · 1000-2000岁 · 金丹化为元婴，神识大增，可形成更强的保命能力。</p></span></a>
<a class="realm-step" href="/cultivation-system/spirit-transformation"><span class="mark">5</span><span><h3>化神期 / Spirit Severing / Transformation</h3><p>下境界 · 数千岁 · 元婴化神，开始触及更高天地法则。</p></span></a>
<a class="realm-step" href="/cultivation-system/void-refining"><span class="mark">6</span><span><h3>炼虚期 / Void Refinement</h3><p>中境界 · 上万年 · 神魂与虚空、天地之力产生更深层联系。</p></span></a>
<a class="realm-step" href="/cultivation-system/body-integration"><span class="mark">7</span><span><h3>合体期 / Unity / Integration</h3><p>中境界 · 数万年 · 元神、法力与肉身高度融合，战力和生存力显著提升。</p></span></a>
<a class="realm-step" href="/cultivation-system/great-ascension"><span class="mark">8</span><span><h3>大乘期 / Mahayana / Great Vehicle</h3><p>中境界 · 十万年以上 · 肉身与法力重铸，接近凡界修炼顶端。</p></span></a>
<a class="realm-step" href="/cultivation-system/tribulation-crossing"><span class="mark">9</span><span><h3>渡劫期 / Tribulation Transcendence</h3><p>中境界 · 十万年以上 · 度过天劫即可飞升仙界，是凡界通往仙界的门槛。</p></span></a>
<a class="realm-step" href="/cultivation-system/true-immortal"><span class="mark">10</span><span><h3>真仙境 / True Immortal</h3><p>上境界 · 长生 · 飞升后的初始仙境，褪去凡胎，掌握仙灵之力。</p></span></a>
<a class="realm-step" href="/cultivation-system/golden-immortal"><span class="mark">11</span><span><h3>金仙境 / Golden Immortal</h3><p>上境界 · 长生 · 金仙不朽，力量和法则理解进一步稳固。</p></span></a>
<a class="realm-step" href="/cultivation-system/taiyi"><span class="mark">12</span><span><h3>太乙境 / 玉仙境 / Taiyi / Jade Immortal</h3><p>上境界 · 长生 · 仙界高阶境界，开始触及更完整的法则体系。</p></span></a>
<a class="realm-step" href="/cultivation-system/da-luo"><span class="mark">13</span><span><h3>大罗境 / Da Luo / Great Luo</h3><p>上境界 · 长生 · 大罗金仙象征不受普通时空束缚的高位仙境。</p></span></a>
<a class="realm-step" href="/cultivation-system/dao-ancestor"><span class="mark">14</span><span><h3>道祖境 / Dao Ancestor</h3><p>上境界 · 永恒 · 仙界最高境，万道之祖，常被设定为大道源流级存在。</p></span></a>
</div>
