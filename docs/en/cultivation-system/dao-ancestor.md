# Dao Ancestor (道祖境, Dàozǔ Jìng)

<div class="entry-layout"><main class="entry-main">

## Summary

**English:** Dao Ancestor belongs to the Upper Realm. Lifespan reference: eternal. The highest immortal stage, an origin-level existence for countless Daos.

**中文：** 道祖境属于上境界，寿元参考为永恒。仙界最高境，万道之祖，常被设定为大道源流级存在。

## Description

**English:** This realm is one step in the fourteen-realm ladder. It changes how cultivators store power, understand the Dao, survive conflict, and interact with the world.

**中文：** 这一境界是十四级修炼阶梯中的关键节点，会改变修士储存力量、理解大道、面对冲突和接触世界的方式。

## Key Characteristics

- **Realm Tier:** Upper Realm / 上境界
- **Lifespan:** eternal
- **Power Shift:** The highest immortal stage, an origin-level existence for countless Daos.
- **Narrative Role:** It marks a new threshold of ability, status, and risk.

## Breakthrough Conditions

**English:** Realize the ultimate personal Dao and connect with world law at an origin level.

**中文：** 证得自身大道极致，并与世界法则产生源流级联系。
