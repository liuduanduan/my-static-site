# Nascent Soul (元婴期, Yuányīng Qī)

<div class="entry-layout"><main class="entry-main">

## Summary

**English:** Nascent Soul belongs to the Lower Realm. Lifespan reference: 1000-2000 years. The Golden Core transforms into a Nascent Soul, greatly strengthening divine sense.

**中文：** 元婴期属于下境界，寿元参考为1000-2000岁。金丹化为元婴，神识大增，可形成更强的保命能力。

## Description

**English:** This realm is one step in the fourteen-realm ladder. It changes how cultivators store power, understand the Dao, survive conflict, and interact with the world.

**中文：** 这一境界是十四级修炼阶梯中的关键节点，会改变修士储存力量、理解大道、面对冲突和接触世界的方式。

## Key Characteristics

- **Realm Tier:** Lower Realm / 下境界
- **Lifespan:** 1000-2000 years
- **Power Shift:** The Golden Core transforms into a Nascent Soul, greatly strengthening divine sense.
- **Narrative Role:** It marks a new threshold of ability, status, and risk.

## Breakthrough Conditions

**English:** Break the core into a soul-form while enduring spiritual transformation and inner demons.

**中文：** 碎丹成婴，需承受神魂蜕变和心魔考验。
