# Cultivation Realms

Cultivation realms describe the ladder from mortal training to immortal transformation. A common sequence is [Qi Refining](/en/cultivation-system/qi-refining), [Foundation Building](/en/cultivation-system/foundation-building), [Core Formation](/en/cultivation-system/core-formation), [Nascent Soul](/en/cultivation-system/nascent-soul), [Spirit Transformation](/en/cultivation-system/spirit-transformation), [Void Refining](/en/cultivation-system/void-refining), [Body Integration](/en/cultivation-system/body-integration), [Great Ascension](/en/cultivation-system/great-ascension), [Tribulation Crossing](/en/cultivation-system/tribulation-crossing), and [Immortal](/en/cultivation-system/immortal).

## Example Terms

- [Golden Core / 金丹](/en/glossary/golden-core)
- [Nascent Soul / 元婴](/en/glossary/nascent-soul)
- [Tribulation / 渡劫](/en/glossary/tribulation)

## Notes

Every novel can rename, reorder, split, or compress realms. Treat this category as a shared reference map rather than a universal law.
