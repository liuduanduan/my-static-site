## 百科摘要

青竹蜂云剑和剑阵体系中的代表性高阶打法，强调成套飞剑、庚金锋锐和阵法围杀。

它把韩立的飞剑从单件法宝推进为可布阵、可控场、可越阶应敌的战斗系统。
