# 关于 NoICU Cultivator

NoICU Cultivator，中文名《逍遥长生录》，是一个面向全球读者的中英双语修仙文化百科。项目目标是把碎片化的修仙、修真、仙侠与中国幻想术语整理成可检索、可引用、可持续扩展的知识库。

## 品牌理念

**No 996, No ICU, Just Cultivation.**

「不加班，不 ICU，只修仙」是一句玩笑，也是一种定位：这个网站希望用轻松的入口解释复杂的修仙体系，让海外读者、中文读者、译者、创作者和研究者都能快速找到可靠参考。

## 内容范围

- [术语词典](/glossary/)：修仙、灵气、金丹、元婴、天劫等核心概念
- [修炼体系](/cultivation-system/)：炼气期到道祖境的十四级境界阶梯
- [经典作品](/classic-works/)：代表性修仙、修真与仙侠小说的类型说明
- [文化背景](/culture/)：道教、内丹术、五行八卦、中国神话与文学流派
