## 常见流派

- 凡人流：慢热成长、资源稀缺、谨慎求生
- 洪荒流：古神、神话谱系、天地大劫
- 无限流：修炼体系与多世界任务结合
- 宗门竞争：弟子、试炼、传承和派系斗争
