# 元婴 / Nascent Soul

## 定义

元婴是修士在更高阶段形成的灵性存在，常被描述为可离体、可保命、可施展神识力量的第二生命核心。

## 详细解释

元婴期修士通常已是宗门长老或一方强者。元婴可以让斗法从肉身层面进入神魂层面，带来夺舍、逃遁、分身和神识攻击等设定。

有的小说把元婴写成清晰的小人，有的则把它处理为灵魂或元神的凝聚形态。
