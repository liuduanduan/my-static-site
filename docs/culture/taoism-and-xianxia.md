# 道教与修仙 / Taoism and Xianxia

道教影响了修仙小说的词汇和意象，例如道、仙、山、符箓、炼丹、内在转化和顺应天地。

## 在小说中的变化

修仙小说并不等同于真实宗教实践。它把部分概念转化为叙事系统：境界变成等级，悟道变成力量，飞升变成剧情门槛。

## 常见意象

- 求道、悟道
- 仙山、洞府
- 符箓、法印和仪式器具
- 内在炼化与长生
- 天劫作为天地压力
