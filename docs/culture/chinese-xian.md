# 中国神话中的仙 / The Chinese Xian

“仙”常被译作 immortal、transcendent 或 celestial being。在小说里，它通常表示超越凡俗死亡和社会秩序的存在。

## 在小说中的用法

修仙小说把仙既写成目标，也写成社会身份。修士可能追求成仙、飞升仙界，或继承古仙遗留的功法和法宝。

## 常见意象

- 远山、洞府与云雾
- 长寿与脱离凡俗
- 仙衣、飞剑、云、鹤、丹药
- 更高世界和更强敌人
