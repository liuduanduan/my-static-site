# 五行 / Five Elements

五行包括木、火、土、金、水，用来描述生成、克制、转化与平衡关系。

## 在小说中的用法

修仙小说常把五行用于灵根、阵法、符箓、丹药、地理、功法和战斗克制。

## 相生

- 木生火
- 火生土
- 土生金
- 金生水
- 水生木

## 相克

- 木克土
- 土克水
- 水克火
- 火克金
- 金克木
