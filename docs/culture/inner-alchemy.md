# 内丹术 / Inner Alchemy

内丹术为修仙小说提供了许多隐喻，如炼化、炉鼎、丹、金丹和身心转化。

## 在小说中的变化

小说常把内在修炼外显为可见机制。[金丹](/glossary/golden-core)可能变成真实能量核心，炼化则变成可计量的进阶过程。

## 常见意象

- 丹与内在精华
- 炉鼎和火候
- 周天运转
- 身体作为小宇宙
- 通过修炼追求长生
