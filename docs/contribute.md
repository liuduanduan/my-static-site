# 贡献指南

欢迎通过 GitHub PR 方式补充词条、修正术语、完善来源和扩展双语内容。

## 贡献原则

- 中文为源语言，英文为翻译语言
- 每个词条需包含中文名、英文名、拼音、分类、摘要、正文、相关词条和参考来源
- 修炼境界需补充寿元、核心特征、突破条件和叙事功能
- 经典作品页只做介绍和设定说明，不收录正文
- 引用资料需注明来源，避免大段复制
